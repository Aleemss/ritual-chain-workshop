# MultiStageAIBounty – Three-Stage Verification with Challenges

This contract implements a **three-stage verification system**: Commit → Reveal → Challenge → Judge. Participants can challenge suspicious submissions before AI judgment.

## How it works
1. **Commit phase**: Participants submit hashed answers
2. **Reveal phase**: Participants reveal their answers
3. **Challenge phase**: Participants can challenge suspicious submissions (stake required)
4. **Resolve challenges**: Owner resolves all challenges
5. **AI judging**: Batch judgment of all submissions
6. **Finalization**: Owner selects winner

## Why multi-stage?
Adds human oversight through challenges before AI judgment.
